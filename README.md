# Anuj Singh - Portfolio Website

My personal portfolio website showcasing my work as an Android Developer.

## Tech Stack

- [Astro](https://astro.build/) - Static Site Generator
- HTML, CSS, JavaScript

## Pages

- **Home** - Introduction and hero section
- **Skills** - Technical skills and expertise
- **Projects** - Open source and Android projects
- **Blogs** - Articles from Substack
- **Videos** - YouTube content
- **Contact** - Social links and contact information

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
